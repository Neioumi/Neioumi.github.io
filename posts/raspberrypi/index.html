Raspberry Piが届いたのでセットアップした

Raspberry Piが届いたー！

Raspberry Piは名刺サイズの小さなLinuxコンピュータです。
11月29日に注文して、到着まで3週間ほど掛かるという事で、首を長くして到着を楽しみにしていました。
いつもの海外通販のように、小包として宅配便の人または郵便局の人から手渡しだろうと思っていたら、あっさりと郵便受けに入っていてちょっぴり拍子抜けw

Raspberry Pi本体（Type B Single Board Computer 512MB）と、クリアケース、そしてRaspberry Pi OSが入った4GB SDカードの3点を購入しました。本体が入っていたピンクのケースも可愛い。
それぞれの価格は、£21.60、£3.99、£9.99。これに送料£4.95を足した、合計£40.53でした。
1ポンド136円換算で5512円。本体だけなら約3000円という安さです！
購入は、RSオンラインにて。

セットアップ

早速、本体をクリアケースに納めました。

普通なら、自分でSDカードにOSをインストールしなければいけないのですが、ものぐさな私はRaspberry Pi OSの入ったSDカードも購入していたので、それを本体に挿入します。

USBがふたつあるので、キーボードとマウスを接続します。

ディスプレイの接続はHDMIで、部屋にあるディスプレイがDVIだったので、事前に購入していた変換ケーブルで接続しました。
LANのところにEthernetケーブルを差します。そのうち無線LAN対応したいところです。
電源はMicro USBなので、とりあえずAndroidタブレット充電用のケーブルを使い回し。
キーボード、マウス、ディスプレイは最近ただの置物になってるWindows機から拝借しています。

特に電源スイッチなどは無いので、電源ケーブルを入れたら起動を始めます。
文字がだらだらだらーっと流れた後、Raspi-config画面となります。


※上の写真では、緑色のオーディオ端子を差していますが、HDMIは音声も付いてるからこれを差す必要はないと有識者の方からご指摘いただきました。

適当にコンフィグを設定して、OSを最新版にアップデートして、パスワード入力、startxでラズベリーのロゴが表示されました！

Minecraft: Pi Editionが面白そう

そもそもなぜ、Raspberry Piに興味を持ったかと言えば、このゲームが一番の理由だったりします。

ブロックで色々なものを作ったり、洞窟を探検したり、色々な遊び方ができるゲームMinecraftが、コマンドで操作できると聞いて、ものすごくやってみたくなりました。
好きなゲームで遊びながらプログラミングやってみれるって楽しそう！
それが、Raspberry Pi向けに移植されるMinecraft: Pi Editionでした。
安くてちっさいパソコンって面白そう、そしてそれでMinecraft: Pi Editionやってみたい、そんな理由で、ついついぽちってしまったわけです。
リリースは年末予定とのこと。
もうクリスマスイブです。まだかなまだかなー！
